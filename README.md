# 3D Beach using modern OpenGL

## Hotkeys

- W/Up      Move up
- S/Down    Move down
- A/Left    Move left
- D/Right   Move right
- Space     Fly up
- LShift    Fly down
- LCtrl     Stop flying
- L         Toggle Alpha LOD
- M         Open tooltip
- C         Switch characters

## Mouse controls

- LMB + movement    Rotate camera
- Mouse wheel       Increase/decrease zoom
